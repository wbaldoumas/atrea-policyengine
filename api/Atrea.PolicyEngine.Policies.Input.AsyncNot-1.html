    Class AsyncNot<T>

    An IAsyncInputPolicy<T> which computes the boolean NOT of a given IAsyncInputPolicy<T>. If a given input policy resolves to Continue or Accept then the item is rejected and Reject is returned, otherwise Continue is returned.
